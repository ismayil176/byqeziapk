# BYQEZI Native Round 7 — Session Persistence Test

Bu roundda native Android login sessiyası yadda saxlanır.

Yoxlama ardıcıllığı:
1. APK-ni quraşdırın.
2. Telefonla OTP giriş edin.
3. Kabinetdə real elanların göründüyünü yoxlayın.
4. App-i tam bağlayın: Recent apps-dən silin.
5. App-i yenidən açın və Kabinetə keçin.
6. Yenidən SMS istəmədən kabinet açılmalıdır.
7. Çıxış düyməsinə basın.
8. App-i bağlayıb açın; artıq login istəməlidir.

Qeyd: Server sessiyası Cloudflare/backend tərəfdə bitərsə, app yenidən login tələb edə bilər. Normal bağlayıb-açma zamanı isə çıxmamalıdır.
