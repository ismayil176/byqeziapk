# Detail Round 1 Scope

Bu round yalnız elan kartına basanda açılan native detail səhifəni əlavə edir.

Dəyişikliklər:
- Home UI və marka/model logic dəyişdirilmədi.
- Kart click → native detail səhifə.
- Detail gallery, thumbnail-lər, qiymət/spec card, satıcı contact card, şikayət qutusu, oxşar elanlar.
- Detail ekranında bottom nav gizlənir, yerinə sticky Zəng edin / WhatsApp barı görünür.

Növbəti round: detail screenshotlarına görə spacing, ölçü və qalereya polish.
