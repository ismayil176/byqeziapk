- byqezi_home_390.png

Daxil olan native komponentlər:
- Header: logo + BYQEZI.AZ + menu
- Search card: Marka / Model autocomplete + Filtrlər + Axtarın
- Demo/canlı status notice
- Banner reklam
- Avtomobil almaq bölməsi
- İki sütunlu elan kartları
- VIP elanlar bölməsi
- Bütün elanlar bölməsi
- Footer: Məxfilik / Haqqımızda / Əlaqə
- Bottom navigation

Sonrakı mərhələ yalnız Home təsdiqlənəndən sonra açılmalıdır.
