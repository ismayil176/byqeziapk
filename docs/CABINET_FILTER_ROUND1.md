# Cabinet Filter Round 1

Bu round-da yalnız kabinetdəki statistika kartlarının filter kimi işləməsi düzəldildi.

Dəyişikliklər:
- Bütün elanlar → istifadəçinin bütün elanları
- Saytda → yalnız active/saytda olan elanlar
- Gözləmədə → yalnız pending/gözləmədə olan elanlar
- Favoritlər → yalnız favorit elanlar
- Aktiv filter vizual olaraq seçili görünür
- Siyahı başlığı və boş state filterə görə dəyişir

- Home logic
- Detail logic
- Login flow
- API endpoint-lər
