# Cabinet UI Round 1 — Pure Visual Polish

Bu round yalnız kabinet görünüşünü yaxşılaşdırır. Funksional məntiqə, Home, Detail, Login, API və bottom nav axınlarına toxunulmayıb.

## Polish edilən hissələr
- Profil kartı: daha böyük avatar, daha səliqəli typography, “Aktiv kabinet” badge-i və balanslı yeni elan düyməsi.
- Dəstək kartı: daha yaxşı spacing, iş saatı badge-i, təmiz çıxış düyməsi.
- Statistika kartları: daha böyük rəqəm, eyni hündürlük, yumşaq radius/shadow.
- Notice kartı: oxunaqlı line-height və pastel fon.
- Mənim elanlarım kartları: daha böyük şəkil, daha yaxşı mətn hierarchy, badge spacing və 42dp action düymələri.

## Qəsdən toxunulmayan hissələr
- Login state
- User listings filter
- Detail səhifə
- Home səhifə
- API endpoint-lər
- Bottom nav logic
