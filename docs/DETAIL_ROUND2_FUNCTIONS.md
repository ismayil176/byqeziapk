# Detail Round 2 — Functional buttons

This round keeps the Home and Detail layout intact and only wires the detail-page controls:

- favorite add/remove on cards and detail header
- image lightbox when tapping the main photo
- next/previous image controls in detail and lightbox
- thumbnail selection updates the main image
- price history modal
- complaint/report modal
- Android back first closes the modal, then returns from detail

No WebView/TWA and no payment changes were added.
